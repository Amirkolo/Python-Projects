# English Dictionary Program

This program allows users to look up the meaning, synonyms, and usage examples of English words using NLTK's WordNet.

## Features

- Look up the meaning of a word.
- Find synonyms for a word.
- View usage examples for a word.

## Getting Started

### Prerequisites

Before running the program, ensure you have Python installed on your system. You'll also need to install the required NLTK packages by running the following command:

```
pip install nltk
```

### Installation

1. Clone the repository:

```
git clone https://github.com/Amirkolo/Python-Projects/edit/main/Dictionary/readME.md
```

2. Navigate to the project directory:

```
cd english-dictionary
```

3. Run the program:

```
python english_dictionary.py
```

## Usage

1. When prompted, enter a word you want to look up.
2. The program will display the meaning, synonyms, and usage examples of the word.
3. To exit the program, type "exit" or "quit".
